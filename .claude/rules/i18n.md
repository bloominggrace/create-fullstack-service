# 이름 정책

- 키는 표시될 한국어 문장을 어순 그대로 옮기며 기능, 구현, 위치 메타는 넣지 않는다.
- 문장이 길어 키를 그대로 옮기기 어려우면 핵심 명사구만 남긴다.

# 정렬 정책

- 키는 알파벳 순서로 정렬한다.

# 예시

| 문장                     | 키                  |
|------------------------|--------------------|
| 전체 동의                  | `allAgree`         |
| 휴대폰번호로 시작하기            | `phoneNumberStart` |
| 서비스 이용을 위해 약관에 동의해 주세요 | `termsAgree`       |
| SMS 인증으로 로그인           | `smsSignIn`        |
